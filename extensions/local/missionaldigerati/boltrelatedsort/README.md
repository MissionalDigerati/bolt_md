Bolt Related Sort
=================

"Bolt Related Sort" is a small [Twig](http://twig.sensiolabs.org) extension built for the [Bolt CMS](http://bolt.cm) to sort through the given relation.

`{{ related_sort(page.related(), 'title', 'asc') }}`
